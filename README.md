# ljtechdotca Twitch Overlay

Twitch overlay for ljtechdotca.

## Authors

- [@ljtechdotca](https://www.github.com/ljtechdotca)

## Features

- Random pokemon through
- Dark/light theme mode

## Run Locally

Clone the project

```bash
git clone https://github.com/ljtechdotca/twitch-overlay.git/
```

Go to the project directory

```bash
cd twitch-overlay
```

Install dependencies

```bash
yarn
```

Start the server

```bash
yarn dev
```

## Tech Stack

**Client:** React, Next

**Server:** Node

## Dependencies

- [sass](https://www.npmjs.com/package/sass)
